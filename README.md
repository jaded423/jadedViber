# JadedViber

🐍 **Live site → [jadedviber.com](https://jadedviber.com)**

Personal brand site of Joshua Brown. Vibe coder. Homelab tinkerer. AI-driven development.

Built end-to-end through dialogue with AI — no boilerplate, no templates. Dracula-themed, cyberpunk-styled, monospace everything.

## Pages

- **[Home](https://jadedviber.com/)** — philosophy, projects, stack, roadmap
- **[Homelab](https://jadedviber.com/homelab.html)** — interactive Proxmox cluster topology
- **[Now](https://jadedviber.com/now.html)** — current focus and active projects

## Stack

- Vanilla HTML / CSS / JS (no framework, no build step)
- Hosted on GitHub Pages
- Custom interactions: terminal block reveal, ASCII snek easter egg, Turing machine binary counter (try typing `turing`)

## Local dev

git clone git@github.com:jaded423/jadedViber.git
cd jadedViber
open index.html        # or any static server
